# Mekanism: Weapons Unofficial

This is an **unofficial** version of the **Mekanism: Weapons** mod, originally created by **oMeranha_**. This project is maintained and updated by Demon7997.

## Features
* **Meka-Tana**: A high-tech katana capable of dealing massive damage.
* **Meka-Bow**: An advanced bow that elevates ranged combat with modular upgrades.

### Modules
**Meka-Bow Modules:**
* **Auto Fire**: Enables continuous, automatic arrow shooting.
* **Draw Speed**: Reduces the time required to fully draw the bow.
* **Gravity Dampener**: Arrows fly straight for improved accuracy.
* **Energy Arrows**: Fires pure energy arrows (no ammunition required).

**Meka-Tana Modules:**
* **Sweeping**: Grants sweeping attacks, hitting multiple targets.

**Common Modules:**
* **Weapon Attack Amplification Unit**: Increases damage (up to 5x, configurable).
* **Looting Unit**: Up to level 3, functioning like the Looting enchantment.

## Additional Features
* **Wireless Charger**: Automatically charges items in your inventory.
* **Curios/Baubles Support**: Visually displays the Meka-Bow and Meka-Tana on your character’s back.
